## Introduction

The default output folder is named `output_pesci`. To better identify each pesci run, it is recommended to specify the output directory with `--outdir` (in short `-o`) and use the `--label1` (`-l1`) and `--label2` (`-l2`) options to label the datasets being compared. 

In the following sections below, we describe the outputs generated by the example command:

```
pesci --matrix1 data/Cragig_matrix_EM.tsv.gz --matrix2 data/Procro_matrix_EM.tsv.gz --clusters1 data/Cragig_cell_id.tsv --clusters2 data/Procro_cell_id.tsv --ortho_pairs data/orthologous_pairs_Procro-Cragig.txt --colbroad broad -l1 Oyster-larva -l2 Flatworm-larva --cores 4 -o output_pesci_larva
```

## Heatmap Figure

The heatmap figure is saved in `output_pesci_larva/Oyster-larva-Flatworm-larva_correlation_scores_matrix.pdf`. The default format for the output figure is a pdf, but it can be changed to svg or png using the `--format` (short `-f`) option (e.g. `--format png`). 

For a description of all available heatmap figure customization options, please see illustrated examples presented in the [Tips and Example Gallery](https://github.com/eparey/pesci/blob/main/wiki/Tips-and-Examples.md#customizing-the-output-figure). 

## Heatmap Raw Data File

The raw data underlying the heatmap figure, i.e. containing the expression similarity scores across cell clusters of the two compared datasets, can be found in `output_pesci_larva/Oyster-larva-Flatworm-larva_correlation_scores_matrix.tsv`. This is a standard tab-delimited table with column and row names, which can be directly loaded in R or python.

## Co-expressed Marker Genes Table

The list of co-expressed marker genes driving expression similarity scores is reported in the table `output_pesci_larva/Oyster-larva-Flatworm-larva_gene_coexpression_table.tsv`.

This is is a tab-separated file with 8 columns, where each row reports a pair of conserved orthologous gene marker. The 8 columns are as follows: Cell_cluster_dataset1 , Cell_cluster_dataset2, Orthologous_marker_Gene_dataset1, Orthologous_marker_Gene_dataset2, Orthologous_Marker_Gene_expression_dataset1 (fold-change), Orthologous_Marker_Gene_expression_dataset2 (fold-change), Expression_co-correlation_score (EC_score), Ortholog_1-1 (yes/no).

By default, marker genes are defined as having fold_change > 1.5 (fold change defined as the expression in cluster of interest over the median of expression across all clusters). This definition can be customized using the options `--min_fc` and `--marker_specificity`. For instance, `--marker_specificity 0.6` will compute fold_change over the 60th percentile of expression across all clusters (instead of expression over the median) and `--min_fc 2` will report only conserved markers with a fold-change > 2 in both datasets. While '`--min_fc` only impact markers reported in the co_expression table, `--marker_specificity` impacts both reported markers and resulting expression similarity scores. 

The EC_score ranges between 0 and 1 and corresponds to the expression co-correlation of the orthologous gene pair.

## Log file

Pesci automatically saves all log printed to screen to a file. This logfile contains useful information regarding a pesci run, including the exact command used, the pesci version as well as the number of retained cells, cluster and orthologous genes. By default, the logfile is saved to `pesci.log` inside the output folder (i.e. `output_pesci_larva/pesci.log` in this example).

> [!NOTE]
> If several pesci commands are run using the same output folder, the log will be appended to the existing logfile in that folder. This can be overridden by specifying a custom logfile location using `--logfile`, for example `--logfile mypath/pesci_logfile.log`. 


## Intermediary outputs

Finally, several intermediary outputs are stored in `files/ `, including :

- The fold change expression matrix (e.g. `output_pesci_larva/files/Flatworm-larva_Procro_matrix_EM.tsv_expr_clusters_norm_all.tsv`)

- The EC_scores for 1-1 orthologs and for many-to-many orthologs (e.g. `output_pesci_larva/files/Oyster-larva-Flatworm-larva_1-to-1-orthologs_correlation_scores.tsv` and `Oyster-larva-Flatworm-larva_orthologs_many_correlation_scores.txt`)

- The list of skipped highly-multigenic orthologous gene families (e.g. `output_pesci_larva/files/Oyster-larva-Flatworm-larva_skipped_ogs.txt`) - usually less than 5 families are skipped, if the number of skipped families is too high the `--max_combin` parameter can be increased (default is `--max_combin 300`)
