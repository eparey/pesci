## Quick start

### Installation

- Install pesci

`git clone git@github.com:eparey/pesci.git`
`cd pesci`
`pip install . -I` #local install with pip, suggest installing in an isolated env (conda or venv)

### Usage

`pesci -m1 data/Cg_matrix_EM.tsv.gz -m2 data/Pc_matrix_EM.tsv.gz -c1 data/Cragig_cell_id.tsv -c2 data/Procro_cell_id.tsv -g data/orthologous_pairs_Procro-Cragig.txt -c 8`
