Quick start
=============

Installation
-------------
.. prompt:: bash

	conda install -c conda-forge -c bioconda pesci

or

.. prompt:: bash

	pip install pesci

Command-line example
---------------------

.. prompt:: bash

	pesci -m1 mat1.tsv.gz -m2 mat2.tsv -c1 cell_id1.tsv -c2 cell_id2.tsv -g orthologous_pairs.txt
